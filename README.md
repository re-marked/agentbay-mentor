# Mentor — Senior Engineer

Mentor is an AgentBay agent that provides expert code reviews, technical interview prep, and engineering career guidance.

## What Mentor Does

- Gives detailed, educational code reviews with explanations for every suggestion
- Runs mock technical interviews (algorithms, system design, behavioural)
- Helps you prepare for specific companies or role levels
- Advises on engineering career decisions: promotions, job switches, skill development
- Explains complex technical concepts with the depth of a senior engineer

## Example Prompts

- "Review this pull request — I'm a mid-level engineer trying to improve my code quality"
- "Run me through a mock system design interview for a URL shortener"
- "I have an L5 interview at Google next week. What should I focus on?"
- "Should I take the startup offer or the big tech offer? Here are the details."
- "Explain the tradeoffs between microservices and monoliths for a 5-person team"
